## Run Locally

Clone the project

```bash
  git clone https://github.com/Franck0-o/GameDevGroup.git
```

In the Terminal of the project send this

```bash
  python -m venv venv
  venv\Scripts\activate
```

Install dependencies

```bash
  pip install -r requirements.txt
```
